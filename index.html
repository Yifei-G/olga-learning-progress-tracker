This is a simple web application where user can convert different type of units like: weight(lb, kg, t), length(ft, m, yd), temperature(C, K, F) and currency(Euro, US dollar and Chinese Yan).
You can input any numeric values from one of the mentioned units and converted into another desi`red unit by clicking the CONVERT button.
Any non-numeric input will not be be allowed and an error message will be displayed in order to inform the user to enter a correct input.
This project has a kawaii design priciple, we give the user a cute cat user experience.

What I have learnt from Olga's converter: This is small size project where I am focusing on putting all my frontend skills into one single piece, which means, I put the HTML, CSS and Javascript into one single project. During the development of this project, I learnt how to use git to control the version of each development.
